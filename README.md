# 🎮 坦克大战 / Tank Battle

一款纯 HTML5 Canvas 坦克大战游戏，经典玩法，简约像素风格。

## 🎯 玩法说明

- **移动**: 方向键 ⬆⬇⬅➡ 或 WASD
- **射击**: 空格键
- **暂停**: P

## ⚡ 特色

- 回合制关卡，击败所有敌人进入下一关
- 随机生成地形（砖墙可破坏，钢墙不可破坏）
- 多种敌人类型（普通、快速、重型）
- 道具系统：加速⚡、连射🔥、护盾🛡、生命❤
- 保护基地旗帜，被敌人击中则游戏结束
- 粒子特效和音效

## 🚀 运行

直接用浏览器打开 `index.html` 即可。
